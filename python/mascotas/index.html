🐾 API de Mascotas

Bienvenido al sistema de gestión de solicitudes de adopción de mascotas.

FastAPI + CRUD + CORS
Ver documentación Ver solicitudes

Endpoints disponibles

POST /solicitudes Crear una solicitud

GET /solicitudes Obtener todas las solicitudes

GET /solicitudes/{id} Obtener una solicitud por ID

PATCH /solicitudes/{id} Actualizar mascota deseada o motivo

PATCH /solicitudes/{id}/estado Cambiar estado

DELETE /solicitudes/{id} Eliminar solicitud
